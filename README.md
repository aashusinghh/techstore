# TechStore

A full-stack e-commerce web application where users can browse products, add items to a cart, sign up, log in, and simulate purchases.

## Features

User signup & login

View product catalog

Add/remove items from cart

Responsive UI

Backend API for product and user operations

## Tech Stack

Frontend: HTML, CSS, JavaScript, React

Backend: Node.js, Express.js

Database: MySql 

## Folder Structure
techstore/
 ┣ frontend/
 ┣ backend/
 ┣ package.json
 ┗ README.md

 ## Setup
git clone https://github.com/aashusinghh/techstore.git
cd techstore
npm install
npm start


Runs at → http://localhost:3000/
